# Ultrasound sensor

## Connector pinout

| Pin | Signal             | Voltage |
| --- | ------------------ | ------- |
| 1   | VCC                | +5.0V   |
| 2   | ULTRASOUND TRIGGER | +5.0V   |
| 3   | ULTRASOUND ECHO    | +5.0V   |
| 4   | GND                | GND     |
