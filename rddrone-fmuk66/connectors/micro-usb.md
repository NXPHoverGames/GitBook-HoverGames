# Micro USB

## Connector pinout

| Pin | Signal | Voltage |
| --- | ------ | ------- |
| 1   | VCC    | +5.0V   |
| 2   | Data-  | +3.3V   |
| 3   | Data+  | +3.3V   |
| 4   | NC     | NC      |
| 5   | GND    | GND     |
